# Real-Time Face Detection System

Bu uygulama, bilgisayar kamerası üzerinden gerçek zamanlı olarak yüz tespiti ve takibi yapmaktadır.

## Özellikler
- **Gerçek Zamanlı İşleme:** OpenCV tabanlı hızlı ve optimize edilmiş tespit.
- **Haar Cascade Algoritması:** Düşük donanımlarda bile yüksek performanslı çalışma.
- **Görsel Bildirim:** Tespit edilen yüzlerin etrafında anlık kare ve etiket gösterimi.

## Teknolojiler
- Python
- OpenCV

## Gereksinimler
```bash
pip install opencv-python
```

## Çalıştırma
```bash
python face_detection.py
```
Uygulamadan çıkmak için kamera penceresi aktifken **'q'** tuşuna basın.
